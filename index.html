깃허브 실시간 서버 연동 가이드

1분 만에 무료 데이터베이스 연결하기

현재 깃허브 페이지(github.io) 환경이 감지되어 가상의 시뮬레이션 모드로 작동 중입니다. 실제 서버에 메시지가 영구 저장되게 하려면 아래 순서대로 본인의 Firebase 주소를 입력해 주셔야 합니다.

1단계: 무료 프로젝트 만들기
Firebase 콘솔에 접속 후 구글 로그인 -> [프로젝트 추가] 버튼을 눌러 새 프로젝트를 생성합니다.
2단계: 데이터베이스 활성화
왼쪽 메뉴에서 Build -> Firestore Database로 이동 후 [데이터베이스 만들기]를 누릅니다. 위치는 서울(asia-northeast3)로 선택하시고, 보안 규칙은 일단 [테스트 모드에서 시작]을 클릭해 활성화해 주세요.
3단계: 웹앱 등록 및 키값 복사
프로젝트 홈 화면 중앙의 웹 아이콘(</>)을 눌러 닉네임을 적고 앱을 등록하면 const firebaseConfig = { ... } 설정 코드가 나타납니다.
4단계: 코드에 붙여넣기 후 깃허브 업로드
현재 코드의 const myFirebaseConfig = { ... } 주석 아래에 발급받은 실제 데이터 객체를 그대로 채워 넣고 저장한 뒤, 깃허브에 다시 푸시하면 실시간 연동이 정상 작동합니다!
